  This folder is a "GeoScope" project web client.
  No installation needed. Just copy this folder to the computer on the net.
  Executive file - "SOAPClient.exe". 
    Start format:
    ========================================================
      SOAPClient.exe ServerURL UserName Password
    --------------------------------------------------------
        where
          ServerURL - url of the project server;
          UserName - project user name (case sensitive);
          Password - user password.
    ========================================================
    remark: for quick start you may edit and use command file
            StartIt.bat


    For co-developers: client source code was designed using Borland Delphi-7
    with RxLibrary components pack. Project file is SOAPClient.dpr
