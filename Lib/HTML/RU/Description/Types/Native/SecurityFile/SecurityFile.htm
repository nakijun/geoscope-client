Файл безопасности.

Файл безопасности это компонент, регулирующий доступ для других объектов. Он содержит список доступных для выполнения операций и их ключи доступа.

Основные пункты:

"список операций и их ключи доступа" - список, отображающий возможные операции и их ключи, имея которые можно выполнить требуемую операцию. Редактируется через контекстное меню списка, пользователем, имеющим права изменения данного файла безопасности.
