Объект карты

Объект является базовым элементом формирования карты. Он может быть разных типов в зависимости от формата используемой первичной карты. Для "Польского" формата объекты подразделяются на три сорта: полилиния, полигон и точка интереса, и могут быть разнообразных типов в каждом сорте, например для точек интереса типы могут быть такие: АЗС, Аптека, Автомойка, Банк и т.д.
