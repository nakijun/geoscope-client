Область извещения для объекта гео-мониторирования.

Данный компонент представляет собой место на карте - регион, при попадании в который "объекта гео-мониторирования" и/или при выходе из него, посылается извещение адресатам, подписанным на события объекта.

Основные пункты:

"Показать" - кнопка, показывающая начало региона на карте.

"Компонент" - кнопка, показывающая компонентный состав данного составного компонента.

"Копировать компонент-визуализацию в карман" - пункт контекстного меню, позволяющий сохранить ссылку на данный компонент, и вставить его в целевой "объект гео-мониторирования".
