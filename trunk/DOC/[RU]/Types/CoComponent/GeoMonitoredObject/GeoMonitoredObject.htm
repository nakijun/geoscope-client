Объект гео-мониторирования.

Данный компонент предназначен для наблюдения за подвижными объектами оборудованными специальными устройства приема информации о местоположении (GPS/GLONASS). Работает совместно с сервером таких устройств - GeoGraph сервером. В него включены механизмы оповещения при входе/выходе в определенные области (места на карте), и оповещения при приближении/удалении других объектов назначенных пользователем.

Основные пункты:

"текущий статус устройства" - показывает текущее состояние связи мобильного устройства с сервером и его статус приема сигналов местоположения (GPS/GLONASS).

"панель свойств устройства"  - кнопка, открывающая панель свойств устройства прикрепленного к объекту. В зависимости от модели устройства эта панель может быть различной.

 "регистрационная информация объекта в GeoGraph сервере" - пункт, показывающий какому GeoGraph-серверу принадлежит данный объект и его регистрационный номер.

"собственник" - кнопка, открывающая панель свойств объекта ассоциированного с данным, если он есть.

"области оповещения при входе и выходе" - список объектов карты, например территорий, вход в которые или выход из них приведет к рассылке адресатам оповещений о произошедшем входе/выходе. Подробнее...

"дистанции оповещения" - список объектов карты, например других мобильных объектов,  приближение к которым на определенное расстояние (удаление на расстояние) приведет к рассылке адресатам оповещений о произошедшем событии. Подробнее...

"адреса для оповещения" - пункт с возможностью редактирования, определяющий кому и каким способом отправлять извещения о произошедших с объектом событиях. Поддерживаются три формата адресов: E-Mail, SMS, MailBox (внутренняя почта проекта). Кнопка "Правка" служит для редактирования адресов этих форматов.

"статус тревоги" - пункт, показывающий текущее состояние пользователя устройства. Тревога подразделяется  на четыре типа: "Нет тревоги". "Слабая", "Сильная", "Критическая". Если устройство выставляет тревогу последних трёх типов, то данный объект заносится в список "Нужна помощь". Посмотреть объекты в этом списке можно через главное меню программы.

 

Контекстное меню панели.

Основные пункты:

"Регистрировать объект на сервере" - пункт регистрации объекта в GeoGraph сервере. Регистрация требуется для того, чтобы вновь созданный объект был известен на сервере, и тогда можно будет использовать мобильное устройство. В этом пункте открывается мастер регистрации, где необходимо будет выбрать сервер, модель объекта, и зарегистрировать его. Подробнее...

"Панель инициализации устройства"  - пункт, открывающий панель для инициализации нового устройства в соответствии с данными текущего объекта и текущего пользователя. При инициализации на устройство отправляется СМС с настройками, и его можно уже будет использовать в пути. Подробнее...

"Копировать компонент-визуализацию в карман" - пункт, копирующий ссылку на текущий объект в карман, для последующего использования.
