Назначение объекту гео-мониторирования областей оповещения.

Инструкция по шагам.

1. Создаем объект типа "Область извещения для объекта гео-мониторирования" или используем уже существующий.

Появляется панель его свойств.

2. В контекстном меню выбираем пункт - "Копировать компонент-визуализацию в карман".

3. Для объекта гео-мониторирования, в списке "Области извещений", открываем контекстное меню, и добавляем из "кармана" область извещения в соответствии с типом извещения ("внешние" - оповещать при выходе из области, "внутренние" - при входе, "Вход/Выход" - в обоих случаях).
