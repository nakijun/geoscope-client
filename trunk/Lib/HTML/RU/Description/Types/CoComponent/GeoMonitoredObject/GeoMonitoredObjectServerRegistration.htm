Регистрация объекта на GeoGraph-сервере.

При регистрации объекта открывается мастер регистрации состоящий из нескольких экранов-шагов.

 

Шаг № 1

Здесь выбирается сервер, где будет зарегистрирован и обслуживаться объект. Далее выбирается тип, т.е. модель внутренней структуры объекта. Далее выбирается "Бизнес модель", т.е. какое именно функциональное назначение имеет объект, основанный на данном типе, например: охранная система, детский маячок. Другими словами, "тип" описывает какого вида устройство используется, а "бизнес-модель" то, какие варианты применения могут быть построены на базе этого устройства.

 

Шаг № 2

На данном шаге вводятся параметры обслуживания объекта. "Geo Space" определяет какому гео-пространству он относится, и соответственно, как географические координаты будут пересчитываться в местные координаты. Поле "Checkpoint interval" определяет интервал проверки соединения сервера с мобильным  устройством. Поле "Distance threshold" определяет порог отправки координат на сервер (например, каждые 30 метров). Раздел "Device" содержит данные, предоставленные самим производителем устройства.

 

Далее будет предложено создать (зарегистрировать) объект на сервере. После регистрации сервер назначит ему внутренний номер, и приступит к его обслуживанию. В конце появится приглашение инициализировать связанное с объектом мобильное устройство.
