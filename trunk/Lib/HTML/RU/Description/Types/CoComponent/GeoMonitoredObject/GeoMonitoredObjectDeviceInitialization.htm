Инициализация устройства.

Инициализация нужна для введение устройства в действие. Раздел "Параметры" - информационный, здесь ничего менять не нужно. В разделе "Инициализация" необходимо ввести "Пароль устройства" прошитый в нём, "Тел. номер устройства" - тот номер, что на сим-карте устройства, "СОМ порт" связи с телефоном передающим СМС и его "Скорость". Завершается инициализация нажатием на кнопку "послать инициал. SMS".

Примечание: этот раздел относится только для автономных устройств. Для устройств на базе сотового телефона с GPS приемником, инициализация происходит по другому, через настройки приложения.
